# 勤怠管理システム

PHP + MySQL で構築したWebベースの勤怠管理システムです。

## 概要

就労支援事業所向けに開発した打刻・集計システムです。
スタッフがブラウザから出退勤を記録し、管理者がログの確認、Excel出力、スタッフ管理ができます。

## 機能

- 打刻画面：スタッフ一覧から選んで出勤・退勤を記録
- ログ確認画面：当日・当月の打刻ログを一覧表示（管理者認証あり）
- Excel出力：当月の勤務表を日付×スタッフ形式でダウンロード
- スタッフ管理：スタッフの追加・削除

## システム構成

| レイヤー | 技術 |
|---|---|
| フロントエンド | HTML / CSS / JavaScript（SPA風） |
| バックエンド | PHP 8.x |
| データベース | MySQL 5.7以降 |
| サーバー | ConoHa Wing（レンタルサーバー） |

## ファイル構成

```
kintai/
├── config.php    　　　　 — DB接続設定
├── get_staff.php         — スタッフ一覧API
├── get_today_status.php  — 本日の打刻状況API
├── stamp.php             — 打刻API
├── index.html            — 打刻画面
└── admin.html            — ログ確認・Excel出力・スタッフ管理画面
└── get_logs.php          — ログ確認API
└── staff_api.php         — スタッフ管理API

```

## セットアップ

1. `config.php` にDB接続情報を入力
2. `create_tables.sql` をphpMyAdminで実行（テーブル作成）
3. 全ファイルをサーバーにアップロード
4. ブラウザで `index.html` を開く

## 使い方

### 打刻画面（index.html）
- スタッフ名ボタンを押す
- 出勤／退勤ボタンを押す
- 未出勤：白、在籍中：緑、退勤済：グレーで色分け表示

### ログ確認画面（admin.html）
- パスワードでログイン
- 当日タブ：当日の打刻一覧
- 当月タブ：月別の出退勤・在籍時間一覧
- 「Excel出力」ボタンで `勤怠_YYYYMM.xlsx` をダウンロード
- スタッフ管理タブ：スタッフの新規登録と削除
